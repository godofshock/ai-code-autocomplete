# AI Code Autocomplete Engine

A neural network-based system that learns code patterns and predicts the next token, simulating a basic code autocomplete system.

## Features

* Custom tokenizer for source code
* Sequence dataset generation
* LSTM-based neural network
* Training pipeline with PyTorch
* Next-token prediction

## Tech Stack

* Python
* PyTorch
* NLP for code

## Status

✅ Completed (v1)

## Goal

To understand how modern code completion systems work by building one from scratch.

## Future Improvements

* Larger dataset (GitHub repos)
* Transformer-based model
* Multi-language support
